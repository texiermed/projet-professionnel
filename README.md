# Formation R et Santé Publique

Médecin généraliste en reconversion vers la santé publique. Ce dépôt contient mon parcours d'autoformation en analyse de données avec R, orienté épidémiologie et santé publique française.

## Contexte

- Thèse en cours : **Predict-R** — évaluation d'un autoquestionnaire numérique de dépistage du risque de maladie rénale chronique en soins primaires
- Mémoire de DES : coordination territoriale et éthique du care en zone sous-dotée (Ariège)
- Objectif professionnel : postes ARS, Santé Publique France, recherche en santé publique

## Compétences techniques

- **R** : tidyverse (dplyr, ggplot2, tidyr), gtsummary, finalfit, epiR, irr, pROC
- **Statistiques** : tests paramétriques/non paramétriques, régression logistique, Kappa de Cohen, Se/Sp/VPP/VPN, Chi2 de tendance
- **R Markdown** : rapports reproductibles
- **Git/GitHub** : versioning, collaboration
